# FellahDz
FellahDz — A web-based agricultural farm management and tracking system designed for Algerian farmers. Features crop management, worker assignments, equipment tracking, and field activity monitoring with role-based dashboards for farmers and workers.

## Features

- Crop Management
- Worker Management
- Equipment & Seeds Inventory
- Field Activity Tracking
- Role-based Dashboards (Farmer & Worker)

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## How to Run

1. Clone the repo
2. Import `database/schema.sql` into MySQL
3. Configure database credentials
4. Run on XAMPP/WAMP
5. Open `http://localhost/FellahDz`
